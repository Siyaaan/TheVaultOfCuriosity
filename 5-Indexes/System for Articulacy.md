# 🧠 Learning System: Become More Articulate
**Status**:: #active  
**Goal**:: Express thoughts clearly and confidently in speech and writing  
**Duration**:: ~2–3 months (repeatable cycle)  
**Review Frequency**:: Weekly  

---

## ✅ Phase 1: Build Foundation (Weeks 1–2)
> Focus: Vocabulary, clarity of thought, input from articulate sources

### 🔤 Vocabulary Expansion
- [ ] Learn 5–10 new words daily using Anki/Vocabulary.com
- [ ] Use each word in 3 original sentences
- [ ] Create a “Word Bank” note for tracking

### 📚 Read High-Quality Material
- [ ] Read 20 minutes/day from:
    - [ ] George Orwell essays
    - [ ] Paul Graham’s *Essays*
    - [ ] *Elements of Style* or *Thank You for Arguing*
- [ ] Highlight and extract one quote/idea per session

### 🧠 Thought Clarity through Journaling
- [ ] Daily 10-minute journal: express a single idea clearly
- [ ] Ask: “What do I really mean?”
- [ ] Tag insights with `#clarity-note`

---

## 🎙️ Phase 2: Practice Speaking (Weeks 3–4)
> Focus: Speaking with precision and presence

### 🎤 Recording & Reflection
- [ ] Speak on a random topic for 1–2 mins and record
- [ ] Listen and note:
  - Clarity?
  - Rambling?
  - Filler words?

### 🗣️ Shadow Great Speakers
- [ ] Choose one TED Talk / Obama / Tyson clip
- [ ] Shadow the speaker line by line
- [ ] Emulate rhythm, tone, and structure

### 💬 Conversation Practice
- [ ] Join a public speaking club or language exchange
- [ ] Avoid filler words during conversation
- [ ] Set 1 clear intention before speaking

---

## ✍️ Phase 3: Writing to Clarify Thought (Weeks 5–6)
> Focus: Turning clear thoughts into articulate writing

### 📝 Short Essay Practice
- [ ] Write 200–500 word essays, 2x/week
- [ ] Follow: Thesis → Supporting Points → Conclusion
- [ ] Use Hemingway or Grammarly for clarity checks

### 🧪 Rewording Exercise
- [ ] Reword an old paragraph 3 different ways
- [ ] Simplify a complex idea in 100 words or less

---

## 🌐 Phase 4: Public Expression (Ongoing)
> Focus: External application of articulateness

### 🎭 Teaching & Explaining
- [ ] Make a 1–2 min explanation video
- [ ] Write one blog-style note per week
- [ ] Explain a complex topic to a friend

### ⚡ Impromptu Speaking Challenge
- [ ] Use a random word generator
- [ ] Speak 1 minute on-the-spot
- [ ] Record and reflect

---
